# UI 模式

虽然 Krawl 使用 TypeScript 开发，带来了不错的类型提示和检查，降低了项目配置和管理出错的可能。但这些过程是枯燥、重复、对新手不友好。

Krawl 提供了强大的 **UI 模式**功能。你可以在漂亮的界面中管理项目中的一切。

## 启动 UI 模式

在项目目录下执行：

::: code-group

```bash [pnpm]
pnpm ui
```

```bash [yarn]
yarn ui
```

```bash [npm]
npm ui
```

:::
